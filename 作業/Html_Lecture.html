HTML 旗標
  
 
旗標的格式
    所有的旗標(指令)都有一固定的格式，姑且稱為 HTML 文法。它們一定由一個"小於符號"開始，，絕對沒有例外。旗標多半是英文單字的縮寫，學習 HTML 只是要記得這些旗標而已。

範例:
(1) <B>是粗體字旗標。
(2) "牛頭"是被粗體字旗標<B>所控制的文字。
(3) </B>是回復旗標。注意，它也是字母 B，只是前面加了一條斜線 /。

成雙成對
    一些主要的 HTML 旗標都是成對出現，它們也都很容易記憶。以下列出三個常用的文字控制旗標：
    粗體(Bold) B <B>粗體(Bold)</B> 粗體(Bold)
    斜體(Italic) I <I>斜體(Italic)</I> 斜體(Italic)
    鉛字體(Typerwriter) TT <TT>鉛字體(Typerwriter)</TT> 鉛字體(Typerwriter)

單身旗標
    現在介紹幾個常用的單身旗標，意思是它們不需要回復旗標。
    <HR>: HR 是 Horizontal Reference 的縮寫，它會在網頁中顯示一條水平的分隔線。
    <BR>: BR 是 Break 的縮寫，它的作用為另起一行，(跳行)。
    <P>: P 是 Paragraph 的縮寫，它的作用為另起一段，(分段)。它的功能和<BR>有點兒類似，不過段落與段落之間的距離較大，意即<P>多跳一行。 雖然<P>是一個單身旗標，不過你仍可以加上回復旗標 ，協助你保持成雙成對的良好文法習慣。

什麼是CSS?
  
 
CSS是Cascading Style Sheets的縮寫
我們中文稱為「串聯樣式表」
  • 它能讓我們更精確的控制網頁版面的文字、背景、字型等...
  • 它能只修正一個CSS文件，便可同時更新眾多的網頁版面外觀及格式。
  • 它可以使html的文件內碼更精簡，縮小檔案下載的速度。
  • 適用於各種作業平台。
 
認識CSS的標準語法
  
 
H3,H5{color : #999999 ; font-family : 細明體}
「H3與H5」我們稱之為"類別選擇器"
意思是說透過這個樣式給「H3與H5」標籤定義
color : #999999(文字灰色)。
font-family : 細明體(字型為細明體)；
也就是說網頁裡所有定義為「H3與H5」標籤的文字，都會變成灰色、細明體。
定義多個標籤相同屬性「H3,H5」就是以「,」號分開
定義標籤多個屬性「color:#999999; font-family:細明體」就是以「;」號分開

 
CSS串聯樣式表的種類
 
 
CSS串聯樣式表的種類有分為幾種，以下說明較常用的排版樣式：
■在<head></head>之間定義多個標籤的樣式↓
<style type="text/css">
<!--
H3 , H5{color : #999999 ; font-family : 細明體,sans-serif}
-->
</style>
■在<body></body>之間定義單一標籤的樣式↓
<H3 style="color : #999999 ; font-family :細明體,sans-serif">內容</H3>

 
範例簡說
  
 
將下方的語法用在網頁的<head></head>標籤之間
<style type="text/css">
<!--
body,table{
color : #999999 ; /*文字色彩*/
font-family : 細明體,sans-serif ; /*文字字型*/
font-size : 9pt /*文字大小*/
}
-->
</style>
這樣一個css就是定義：
body(整個網頁)和 table(表格) 內的文字為灰色、細明體、9pt的大小。
使用範例你的網頁不可以是空白的,一定要包含文字內容。
 
CSS串聯樣式表教學『選擇器運用』
  
 

之前我們有提到何謂選擇器，例如：table{font-size:9pt}裡面的table就稱為選擇器，就是選用table(表格)在{}裡定義樣式，這個選擇器當然你可以自己設定為別的名稱，讓網頁指定的地方來使用，或者你想每行用不同的文字顏色或不同的表格顏色，沒問題，用自訂選擇器一定讓你更容易掌握。

種類一：CLASS="名稱"

<style type="text/css">
<!--
.ABC{
color : #cc6699 ; /*文字色彩*/
font-size : 9pt /*文字大小*/
}
.DEF{
color : #336699 ; /*文字色彩*/
letter-spacing : 3pt;/*字距*/
font-size : 9pt /*文字大小*/
}
-->
</style>
</head><body>
<p class="ABC" >這裡的文字是粉色9pt的大小</p>
<p class="DEF" >這裡的文字是藍色9pt字的距離4pt的大小
<input class="ABC" type="text" name="T1" size="20" value="這裡是表單">
<table class="DEF" width="182"> <tr><td>這裡是表格</td></tr> </table>

 

種類二：ID="名稱"
還有一種方式就是使用ID，其實這跟上方的意思是相同的，CLASS選擇器是在前方加上小數點，不同的是ID是在前方加上#，來看下面的範例：可以試試看是不是跟class選擇器範例結果是一樣的
#ABC{
color : #cc6699 ; /*文字色彩*/
font-size : 9pt /*文字大小*/
}
#DEF{
color : #336699 ; /*文字色彩*/
letter-spacing : 3pt;/*字距*/
font-size : 9pt /*文字大小*/

 

 
CSS串聯樣式表教學『外部檔案』
 
 

CSS的好處第二項：它能讓我們只修正一個CSS文件，便可同時更新眾多的網頁版面外觀及格式。

連結樣式表和匯入樣式表的分別
匯入(import)樣式表和下面介紹的LINK連結樣式表功能是相同的，只是運作方式不同。如果HTML文件以import的方式匯入樣式檔案，那瀏覽器在讀取HTML文件時，會將樣式表複製一份到HTML文件內；如果使用連結樣式表的方式(LINK)，瀏覽器會再HTML文件內的標籤需要格式設定時，才會向所連結的樣式表索取樣式。

連結樣式表

步驟一、將下方語法複製到記事本，放置在與網頁同一個目錄內
存檔時檔名輸入style.css(就是附檔名格式為.css)
body,table
{color : #999 ; /*文字色彩*/
font-family : 細明體 ; /*文字字型*/
font-size : 9pt; /*文字大小*/
letter-spacing : 2pt ;/*字間距離*/
line-height : 150% ;/*設定行高*/
}
A:link /*超連結*/
{color:#c69;/*文字色彩*/
text-decoration:none;/*刪除連結線*/
}
A:visited /*瀏覽過的超連結*/
{color:#c69;/*文字色彩*/
text-decoration:none;/*刪除連結線*/
}
A:active /*按下連結*/
{color:#c69;/*文字色彩*/
text-decoration:none;/*刪除連結線*/
}
A:hover /*滑鼠移至連結*/
{color:#c30;/*文字色彩*/
border-top : 1px solid #000 ; /*上框*/
border-bottom : 1px solid #000 ; /*下框*/
}

只細看一下少了甚麼？發現了嗎？就是<style type="text/css"> </style>
存為獨立css檔案的時候一定要去掉這標籤

步驟二、在你要連結這css樣式所有網頁的<head></head>標籤中加入這一行

<link rel="stylesheet" type="text/css" href="style.css">

 

匯入樣式表

步驟一、跟上方連結樣式表相同
步驟二、在你要匯入這css樣式所有網頁的<head></head>標籤中加入
<style type="text/css">
<!--
@import url("abc.css");
-->
</style>
